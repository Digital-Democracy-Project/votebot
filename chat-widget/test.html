Page Context Modes

In Mobile App Mode, you explicitly pass the page context. This is ideal for mobile apps or when you have full control over the context.

Embedding Code

<script>
    window.DDPChatConfig = {
        wsUrl: 'wss://api.digitaldemocracyproject.org/votebot/ws',
        pageContext: {
            type: 'bill',
            id: 'HR-1',
            title: 'One Big Beautiful Bill Act',
            jurisdiction: 'US'
        }
        // welcomeMessage is auto-generated based on context
    };
</script>
<script src="https://api.digitaldemocracyproject.org/widget/ddp-chat.min.js"></script>

In Website Auto-Detect Mode, the widget automatically detects the page context from:

  • Data attributes: data-ddp-type, data-ddp-id, data-ddp-title
  • JSON-LD structured data
  • URL patterns (e.g., /bill/HR-1, /legislator/john-smith)
  • Meta tags (og:title, og:type)

Add Data Attributes to Your Page

<body data-ddp-type="bill"
      data-ddp-id="HR-1"
      data-ddp-title="One Big Beautiful Bill Act"
      data-ddp-jurisdiction="US">
  ...
</body>

Or Use JSON-LD

<script type="application/ld+json">
{
  "@context": "https://schema.org",
  "@type": "Legislation",
  "name": "One Big Beautiful Bill Act",
  "identifier": "HR-1"
}
</script>

Embedding Code (Auto-Detect)

<script>
    window.DDPChatConfig = {
        wsUrl: 'wss://api.digitaldemocracyproject.org/votebot/ws',
        autoDetect: true  // Automatically detect context from page
    };
</script>
<script src="https://api.digitaldemocracyproject.org/widget/ddp-chat.min.js"></script>

Welcome Message Preview

The welcome message is automatically personalized based on page context:

Bill context:
"Welcome! I can answer detailed questions about One Big Beautiful Bill Act (HR-1). You can also ask me about other bills, legislators, or Digital Democracy Project in general."
Legislator context:
"Welcome! I can answer questions about Sen. John Smith, including their voting record, sponsored bills, and positions. You can also ask me about other legislators or legislation."
General context:
"Welcome! Ask me anything about legislation, legislators, or civic engagement."

JavaScript API

// Update context dynamically (e.g., when user navigates)
DDPChatWidget.setPageContext({
    type: 'bill',
    id: 'HR-1',
    title: 'One Big Beautiful Bill Act'
}, true);  // true = show new welcome message

// Generate a welcome message for a context
var msg = DDPChatWidget.generateWelcomeMessage(context);

// Get current context
var ctx = DDPChatWidget.getPageContext();
