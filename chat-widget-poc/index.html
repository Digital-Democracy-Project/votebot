VoteBot Chat Widget

A proof-of-concept streaming chat interface for the Digital Democracy Project.

This widget connects to the VoteBot backend via WebSocket for real-time streaming responses.

  • Real-time streaming responses
  • Conversation history
  • Citation display
  • Confidence scoring
  • Human handoff detection
  • Page context awareness

Note: Make sure VoteBot is running on localhost:8000
